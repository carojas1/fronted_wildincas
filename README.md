# Frontend Wild Incas - SIMOT v2.0

Frontend React/Vite para el sistema hotelero Wild Incas.

## Ejecutar

```bash
npm install
npm run dev
```

Para Vercel configurar la variable:

```text
VITE_API_URL=https://URL-DE-TU-BACKEND/api
```

Valores recomendados en Vercel:

- Framework Preset: `Vite`
- Root Directory: `./`
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
